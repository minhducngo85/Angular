Class and Style Binding

Class binding is used to bind the data from component to HTML class property
Syntax: <HTMLTag [class]="component variable holding class name">

This paragraph class comes from *myClass* property

Style binding is used to bind the data from component into HTML style property.
Syntax: <HTMLTag [style.STYLE]="component data">

Text color is styled using style binding

Two Way Data Binding

NgModel

NgModel is a standalone directive. ngModel directive binds form control to property and property to form control.
Systax: <HTML [(ngModel)]="model.name" />
Example: <input type="text" [(ngModel)]="model.name" />
