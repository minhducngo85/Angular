Web workers enables JavaScript application to run the CPU-intensive in the background so that the application main thread concentrate on the smooth operation of UI.

Create a new Angular application using below command −

1.
cd /go/to/workspace
ng new web-worker-sample

2.
Run the application using below command −

3.
cd web-worker-sample
npm run start

4.
Add new web worker using below command −
ng generate web-worker app
